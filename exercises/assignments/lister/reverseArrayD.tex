Write the function
\lstinline{reverseArrayD : arr:'a [] -> unit}, which reverses the
order of the values in \lstinline{arr} using a while-loop to overwrite
its elements. For example, the program
\begin{verbatim}
let aa = [|1..5|]
reverseArrayD aa
printfn "%A" aa
\end{verbatim}
should output \lstinline{[|5; 4; 3; 2; 1|]}.
